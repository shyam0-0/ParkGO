Phase 1 — MVP (Basic Working Demo)

Goal: Show the core idea working.

Features

Driver can:

View parking spots on map

Click spot to see price

Book parking

Owner can:

Add parking location

Set price per hour

System Capabilities

Parking stored in database

Markers appear on map

Booking stored in database

Simple booking confirmation

Tech Used

Frontend

React

TailwindCSS

Leaflet Map

Backend

Flask API

Database

SQLite

Demo Flow

Owner adds parking

Parking appears on map

Driver clicks marker
